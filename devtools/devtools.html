DevTools Presentation

There are many uses for DevTools as a Developer, not limited to:

  • debugging/troubleshooting
  • inspecting and editing HTML/CSS
  • optimization

The different ways to open DevTools.

  • 3 dots > More tools > Developer Tools
  • Right click > Inspect
  • Select an element on page > Right click > Inspect

Panels

  1. Elements: review and change DOM and CSS
  2. Sources
  3. CSS Overview: Summary of CSS used. Check color contrast
  4. Lighthouse: check peformance, accessibility, best proactices and SEO in movile and desktop views
  5. Several other panels
